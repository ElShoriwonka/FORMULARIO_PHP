----------------------------------
INSTRUCCION DE INSTALACION
----------------------------------

PASO 1

1.1 DESCARGAR EL INSTALADOR DE XAMPP EN LA PAGINA OFICIAL, LA CUAL ES LA SIGUIENTE URL: https://www.apachefriends.org/es/index.html(en mi caso use la version para windows pero esta disponible para otros sistemas operativos).

PASO 2

2.1 UNA VEZ INSTALADO XAMPP, IR A LA RUTA DE INSTALACION (DISCO LOCAL C o la ruta que se especifico en el momento de instalar xampp).
2.2 CON LA CARPETA DE INSTALACION DE XAMPP ENCONTRADA IR AL SIGUIENTE DIRECTORIO C:\xampp\htdocs\, QUE ES DONDE DE ALOJAN LOS PROYECTOS CREADOS.

PASE 3

3.1 DESCARGAR EL PROYECTO DE GITHUB DE LA SIGUIENTE URL: https://github.com/ElShoriwonka/FORMULARIO_PHP.git
3.2 LUEGO, DESCOMPRIMIR EL PROYECTO Y MOVERLO A LA CARPETA DE HTDOCS DE XAMMM, EL CUAL DEBDE DE ASI: C:\xampp\htdocs\FORMULARIO

PASO 4

4.1 AL EJECUTAR XAMPP DAR A LA OPCION DE "START" DE MYSQL, DONDE SE DEBERA IMPORTAR LA BASE DE DATOS DEL PROYECTO LLAMADA "formulario", LA CUAL CONTIENE LAS TABLAS A UTILIZAR CON SUS DATOS YA INGRESADOS.
4.2 REGRESAR AL PANEL DE CONTROL DE XAMPP, DONDE MOSTRARA POR DEFECTO UNA PAGINA(http://localhost/dashboard/), LA CUAL REEMPLAZAR POR: http://localhost/FORMULARIO/views/index.php, ESTO MOSTRARA EL PROYECTO REALIZADO.
4.3 CON ESTO YA DEBE DE PODER CARGAR EL PROYECTO DE FORMA CORRECTA.
